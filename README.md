Detect, understand, and fix production errors faster with AI.

---

## 🌍 Live Demo

👉 https://crwatch.vercel.app

---

## 🧠 What is CRWatch?

CRWatch is a lightweight crash monitoring tool that helps developers:

* 📡 Capture errors from frontend & backend
* 🧹 Clean and structure stack traces
* 🤖 Use AI to explain what went wrong
* ⚡ Debug issues faster without digging through logs

---

## ✨ Features

* Real-time error tracking
* AI-powered error explanations
* Clean and readable stack traces
* Simple integration
* Built for indie devs & SaaS builders

---

## ⚙️ Tech Stack

* Next.js
* Supabase
* OpenAI API
* Vercel

## 🚀 Getting Started

Clone the repo:

```bash
git clone https://https://github.com/MKhoi28/cronjob-monitor
cd cronjob-monitor
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open http://localhost:3000

---

## 🔑 Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
GEMINI_API_KEY=your_key
```

---

## 🧪 Example Usage

Initialize monitoring:

```js
initMonitor({
  apiKey: "YOUR_API_KEY"
});
```
